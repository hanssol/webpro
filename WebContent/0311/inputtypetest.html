type : 입력하는 데이터의 형태
			text : 한줄입력
			radio : 여러개 중에서 한개를 선택
			checkbox : 여러개선택 또는 한개선택 또는 선택이 없을 수 있다.
			hidden : 화면에서는 보이지 않는다. 서버로 값이 전송되어진다.
			image : 전송버튼역할을 한다. submit과 같은 역할
			button : 이벤트를 부여해서 전송역할을 한다.
		name : 서버로 전송시 입력한 데이터로 보내기 위한 속성
		textarea  태그(요소)
		button 태그(요소)
			type을 지정하지 않으면 submit(전송)역할을 수행한다.
