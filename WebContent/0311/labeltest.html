인라인 요소 : 필요한 만큼만 차지한다. 줄이 안바뀐다.
		ex) span , a, label, input
		크리를 지정할 수 없다.
	블럭요소 : 자동으로 줄바꿈, 한줄 전체를  차지한다.
			크기를 지정할 수 있다.
		ex) p, div, pre, hr, table, form
		
	인라인 요소의 크기를 지정하기 위해서 display속성을 이용한다.
	display : inline-block;
	display : none
	display : inline;
	display : block;
